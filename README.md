# Lorem Panel — VS Code & Cursor Extension

A simple extension that opens a panel displaying lorem ipsum text with buttons to switch between different variants. Works in both **VS Code** and **Cursor**.

---

## Features

- Opens a dedicated panel via the Command Palette
- Displays lorem ipsum text in a styled card
- Four text variant buttons: **Classic**, **Short**, **Long**, **Cicero**
- Text fades smoothly when switching variants
- Fully themed using VS Code CSS variables (adapts to any color theme)

---

## Requirements

- [Node.js](https://nodejs.org/) v18 or later
- [npm](https://www.npmjs.com/) (bundled with Node.js)
- VS Code 1.85+ **or** Cursor (any recent version)

---

## Setup & Install

### 1. Clone or open the project

```bash
cd CursorVscodeExtension
```

### 2. Install dependencies

```bash
npm install
```

This installs TypeScript, the VS Code type definitions, and ESLint.

---

## Development (Run without packaging)

### 3. Compile the TypeScript source

```bash
npm run compile
```

Or start the file watcher to recompile on every save:

```bash
npm run watch
```

### 4. Launch the extension in a new window

In **VS Code** or **Cursor**, press **F5** (or go to **Run → Start Debugging**).
This opens an **Extension Development Host** window with the extension loaded.

### 5. Open the panel

In the Extension Development Host window, open the Command Palette:

- **macOS**: `Cmd+Shift+P`
- **Windows / Linux**: `Ctrl+Shift+P`

Type and select:

```
Lorem Panel: Open Lorem Panel
```

The panel opens with lorem ipsum text and four buttons at the top. Click any button to switch text variants.

---

## Build & Package (`.vsix`)

To create an installable `.vsix` file:

### Install the packaging tool

```bash
npm install -g @vscode/vsce
```

### Package the extension

```bash
vsce package
```

This produces a file like `lorem-panel-0.0.1.vsix` in the project root.

---

## Install the `.vsix` in VS Code or Cursor

### VS Code

```bash
code --install-extension lorem-panel-0.0.1.vsix
```

Or via the UI: **Extensions → ··· menu → Install from VSIX…**

### Cursor

```bash
cursor --install-extension lorem-panel-0.0.1.vsix
```

Or via the UI in Cursor: **Extensions → ··· menu → Install from VSIX…**
